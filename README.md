### 숫자야구 알고리즘
게임 이론의 적대적 상대를 상정하고 최악의 경우를 선택하더라도 내가 받는 피해를 최소화 하는 방향으로 진행.
제시할 수 있는 전체 집합에서 가장 엔트로피가 적어지는 방향으로 질문을 구성.
결과에 따라서 모든 부분 집합과 비교하여 가장 분할 구역이 낮은 구역을 선택하여 질문을 제작.

### 택배 상자 꺼내기
단순하게 해당 모양의 박스를 2차원 배열로 쌓아올리는 방법을 생각해서 일치하는 숫자가 있으면 answer의 값을 올렸습니다.

### 서버 운영 개수 구하기
전체 운영 시간에 대해 늘어난 서버 개수를 구하고, 가장 중요한 부분인 해당 서버 이용 가능 시간을 현재부터 과거까지의 차이를 계산하여 값을 구했습니다.

### 비디오 플레이어
mm:ss 포맷의 문자열을 초로 환산하여 계산을 한 후, 다시 mm:ss로 변환하여 문제를 풀었습니다. 시작 위치 검증 -> 이전/다음 명령어 처리 -> 엣지 케이스 처리 순으로 구했습니다.
